# 📓 Daily Journal

A terminal-based Daily Journal application built with Python. This project allows users to securely log in, write journal entries, view previous entries, and remove unwanted entries while storing data locally.

## ✨ Features

- 🔐 User login system
- 👤 User registration
- 📝 Write journal entries
- 📖 View saved entries
- 🗑️ Delete journal entries
- 💾 Local file storage
- 📋 Menu-driven interface

## 🛠️ Technologies Used

- Python 3
- JSON
- File Handling
- Functions
- Modules

Clone the repository:
